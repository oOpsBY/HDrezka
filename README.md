> **Мощное расширение для браузера**, которое превращает HDrezka в полноценную базу с рейтингами (Кинопоиск, IMDb) и дает удобный инструмент для скачивания видео и субтитров. Всё это — в чистом, минималистичном интерфейсе, который не режет глаз.

## ✨ Ключевые возможности

### 🎯 Продвинутая система рейтингов
*   **Мгновенные рейтинги HDrezka** — отображаются прямо на карточках фильмов и сериалов.
*   **Интеграция Кинопоиска и IMDb** — рейтинги подтягиваются и отображаются в аккуратных цветных блоках в углу карточки.
*   **Цветовая индикация** — оценки подсвечиваются разными цветами для быстрой визуальной оценки:
    *   🟢 **Зеленый** (7.0+) — отлично
    *   🟠 **Оранжевый** (4.0–6.9) — средне
    *   🔴 **Красный** (ниже 4.0) — так себе
*   **Детализация** — для каждого рейтинга показывается количество голосов в сокращенном формате (1.2K, 1.5M).

### ⬇️ Умный загрузчик видео и субтитров
*   **Скачивание в один клик** — работает, когда плеер активен. Просто нажми кнопку в попапе расширения.
*   **Автоопределение качества** — расширение анализирует видеопоток и показывает разрешение (720p, 1080p, 4K) прямо в интерфейсе.
*   **Субтитры** — автоматически находит все доступные языки субтитров прямо из плеера HDrezka. Можно скачать любой набор одним нажатием.

### 🚀 Технологии и производительность
*   **Умный кэш (Smart Cache)** — все данные рейтингов сохраняются локально на **48 часов**. Страницы грузятся мгновенно, а нагрузка на сайт сводится к нулю.
*   **Никакой слежки** — расширение работает полностью локально. Никакие данные не отправляются на сторонние серверы. Прозрачность — наш приоритет.
*   **Минимализм** — интерфейс вписан в дизайн сайта так, что выглядит как родной.

## 🛠 Установка

### Вариант 1: Из официальных магазинов (рекомендуется)
*   **🦊 Opera Add-ons:** [Добавить расширение]() (ссылка появится после публикации)
*   **🌐 Chrome Web Store:** [Установить из магазина]() (ссылка появится после публикации)
*   **🧭 Microsoft Edge Add-ons:** [Скачать для Edge]() (ссылка появится после публикации)

### Вариант 2: Ручная установка (режим разработчика)
Если хотите протестировать самую свежую версию до официального релиза:
1.  Скачайте архив с последней версией из [раздела Releases](ссылка-на-релизы).
2.  Распакуйте архив в отдельную папку (например, `C:\Extensions\hdrezka-rating`).
3.  Откройте браузер и перейдите на страницу расширений:
    *   **Chrome/Edge/Brave:** `chrome://extensions/`
    *   **Opera:** `about://extensions/`
4.  Включите **«Режим разработчика»** (Developer mode).
5.  Нажмите кнопку **«Загрузить распакованное расширение»** (Load unpacked) и укажите путь к распакованной папке.

## 🧰 Как пользоваться (быстрый старт)

1.  **Установите** расширение одним из способов выше.
2.  Перейдите на сайт HDrezka — **любая страница с фильмами/сериалами** сразу отобразит рейтинги.
3.  **Для скачивания видео/субтитров:**
    *   Откройте страницу с фильмом и **запустите плеер**.
    *   Нажмите на иконку расширения в правом верхнем углу браузера.
    *   Переключитесь на вкладку **«Скачать»**.
    *   Нажмите **«Показать качество»**, чтобы убедиться, что видео распознано.
    *   Выберите язык субтитров (если доступны) и нажмите **«Скачать видео»**.

## 🔒 Политика конфиденциальности

Мы серьезно относимся к вашей приватности. Всё, что вы видите — остаётся на вашем компьютере.
*   **Локальная работа:** Рейтинги кэшируются в вашем браузере через Chrome Storage API.
*   **Никаких серверов:** Расширение не отправляет никаких данных на внешние серверы.
*   **Прозрачность:** Исходный код открыт и всегда доступен для проверки.
